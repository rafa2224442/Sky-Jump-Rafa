# Sky-Jump-Rafa
Sky Jump adalah sebuah game 2D buatan sendiri yang dibuat menggunakan bahasa pemrograman Python dengan library Pygame. Dalam game ini, pemain mengendalikan sebuah karakter yang harus terus melompat dari satu platform ke platform lainnya agar tidak jatuh. Platform akan bergerak turun saat pemain melompat lebih tinggi, sehingga memberikan efek scrolling ke atas seperti pada game Doodle Jump.

# Cara memmainkan
1.Gunakan tombol A untuk bergerak ke kiri
2.Gunakan tombol D untuk bergerak ke kanan
3.Pemain akan melompat otomatis setiap kali menginjak platform
4.Jika pemain keluar dari sisi kiri layar, pemain akan muncul kembali dari sisi kanan (dan sebaliknya)
5.Tujuan utama adalah bertahan selama mungkin dan meraih skor setinggi mungkin
6.Game berakhir jika pemain terjatuh keluar layar bagian bawah

# Hasil
Game berhasil dijalankan dengan fitur sebagai berikut:
✔ Player bisa bergerak kiri-kanan
✔ Sistem gravitasi dan lompatan bekerja baik
✔ Platform dibuat secara acak dan terus muncul selama permainan berlangsung
✔ Sistem scrolling menambah kesan naik ke atas
✔ Skor terus bertambah sesuai ketinggian lompatan
✔ Tampilan game dilengkapi background, sprite pemain, dan warna platform
✔ Ada tampilan Game Over beserta skor akhir
